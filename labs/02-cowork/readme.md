# Copilot Cowork Labs

Hands-on labs for Copilot Cowork: tenant setup, the execution model, custom Agent Skills, and packaged plugins with MCP connectors. Work through them in order, each lab builds on the previous one.

| # | Lab | What you build |
|---|-----|----------------|
| 1 | [Cowork Setup and Extensibility](./lab-01-cowork-setup.md) | A Cowork-ready tenant, a first delegated task, and a map of the skills vs plugins extensibility surfaces |
| 2 | [Build Your First Cowork Skill](./lab-02-cowork-skills.md) | A custom Agent Skill, created once with the guided flow and once by hand-authoring `SKILL.md` |
| 3 | [Build Your First Cowork Plugin](./lab-03-cowork-plugins.md) | A plugin package with three skills and a Microsoft Learn MCP connector, deployed from the admin center |

## Prerequisites

- A Microsoft 365 tenant with tenant admin access, for example from the [Microsoft 365 Developer Program](https://developer.microsoft.com/en-us/microsoft-365/dev-program)
- An active Microsoft 365 Copilot license and Cowork availability in the tenant
- Usage-based billing configured for Copilot Credits
- Visual Studio Code for the skill and plugin authoring exercises

> Authoring and testing custom skills and plugins consumes Copilot Credits. Set spending limits before you start.

Pairs with [Module 2: Cowork](../../demos/02-cowork/readme.md). The lab content is adapted from the [Copilot Developer Camp](https://microsoft.github.io/copilot-camp/pages/copilot-cowork/00-cowork-setup/) Cowork pathway.
