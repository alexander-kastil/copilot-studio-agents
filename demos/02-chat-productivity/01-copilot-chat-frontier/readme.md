# Copilot Chat & Microsoft Frontier

Copilot Chat is the always-on AI chat experience in Microsoft 365. You ask a question or describe a task in plain language, and Copilot drafts, summarizes, or answers using web content and (with a license) your work content in Microsoft Graph. This topic covers the chat experience, how it changes when you add a Microsoft 365 Copilot license, how to stay on top of work with Copilot Tasks, and how the Microsoft Frontier program unlocks early-access features.

---

## Copilot Chat vs. Microsoft 365 Copilot

Everyone in a commercial tenant gets Copilot Chat for free, grounded in the web. Adding a Microsoft 365 Copilot license grounds the same chat in your organization's data and unlocks the deeper features.

| | Copilot Chat (free) | Microsoft 365 Copilot (licensed) |
|---|---|---|
| Grounding | Public web content | Your emails, chats, files, and meetings via Microsoft Graph |
| In-app help | Standalone chat | Embedded in Word, Excel, PowerPoint, Outlook, and Teams |
| Prebuilt agents | Limited | Researcher, Analyst, and the coach agents |
| Controls | Basic | Copilot Control System and Copilot Analytics |

Start with Copilot Chat for quick drafts and answers. Add the license when you need Copilot to reason over your own work content and act inside the apps you already use.

## Where to Access Copilot Chat

- Browser: [m365.cloud.microsoft](https://m365.cloud.microsoft)
- Desktop: the Microsoft 365 Copilot app for Windows and Mac
- Mobile: the Microsoft 365 Copilot app for iOS and Android
- Inside Teams, Outlook, Word, Excel, and PowerPoint

## Copilot Tasks: Stay on Top of Work

Copilot helps you capture and track work, not just answer questions. Two capabilities matter here.

- Action items: ask Copilot to pull the to-dos and owners out of a meeting recap or a long email thread, so nothing slips.
- Scheduled prompts: set a prompt to run on a recurring schedule (for example a Monday status summary), and Copilot delivers the result without you asking each time.

Long-running work shows its status right in the Microsoft 365 Copilot app, and on Windows you can watch task progress from the taskbar. Recurring automation across your apps is covered in depth in [Module 3: Automation & Plugins](../../03-cowork/03-automation/readme.md).

## The Microsoft Frontier Program

Frontier is Microsoft's early-access program for the newest Copilot innovations. Enrolling opts your tenant into preview features before general availability, so your organization can try, shape, and plan for what is coming.

Frontier is where capabilities like Copilot Cowork first appear. Preview features are subject to change and may carry different licensing, so treat them as evaluation, not production. Enrollment and pay-as-you-go setup are covered in [Module 1: Pay as you go licensing](../../01-intro/04-licensing-setup/readme.md#pay-as-you-go-licensing).

## Practical Scenarios

| Scenario | How Copilot Chat Handles It |
|---|---|
| Catch up fast | Summarize a 40-message Teams thread into five bullets before a meeting |
| Draft with context | Write a reply that pulls the latest numbers from a file you can access |
| Extract action items | Turn a meeting recap into a checklist of tasks and owners |
| Recurring reporting | Schedule a weekly prompt that compiles your team's highlights |
